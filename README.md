# 3D Car Racing Arena using OpenGL

This project is a **3D car racing and shooting game** developed using Python and OpenGL. The player controls a car, avoids obstacles, shoots enemies, and reaches the finish line before enemy cars do.

## Features
- 3D player car and enemy cars
- Bullet shooting system
- Enemy movement and enemy shooting
- Obstacles on track
- Weather modes (Day / Night / Rainy)
- Camera switching (First-person / Third-person)
- Cheat mode with auto-drive and auto-fire
- Boost mode for temporary speed increase
- Score, lives, and bullet tracking

## Controls
- `W / S` → Move forward / backward  
- `A / D` → Rotate car  
- `Left Click` → Shoot bullet  
- `Right Click` → Change camera view  
- `1 / 2 / 3` → Change weather  
- `C` → Toggle cheat mode  
- `Space` → Activate boost  

## Game Logic
- Player wins by reaching the finish line  
- Game over if enemy reaches first  
- Game over if player loses all lives  

## Technologies Used
- Python  
- OpenGL (PyOpenGL)  
- GLUT
